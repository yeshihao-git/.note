# C++11标准

1. **自动类型推导机制**
	- auto：根据 初始化表达式 推导类型，会计算表达式，丢弃顶层const和引用
	- decltype：编译时推导表达式类型，不会计算表达式，保留顶层const和引用
2. **范围for循环**：简化容器遍历
3. **智能指针**：RAII思想的体现，用类的生命周期自动化管理资源
4. **[[lambda表达式]]**：支持匿名函数
5. **列表初始化**：统一了不同场景初始化语法（使用 ={}= 列表初始化）
6. **移动语义**：减少拷贝开销，提升性能
7. **新容器** ：std::unordered_map、std::unordered_set
8. **标准的线程库**
9. **nullptr**：cpp11关键字，NULL是C语言中的宏（定义为0，无法与整数0区分），导致函数重载的歧义问题 
```cpp
int* p1 = NULL;    // 合法但可能有歧义
int* p2 = nullptr; // 更安全的空指针赋值

// 重载场景下的区别
void func(int x) { ... }
void func(char* p) { ... }

func(NULL);  // 调用 func(int)，可能不符合预期
func(nullptr); // 正确调用 func(char*)
```
