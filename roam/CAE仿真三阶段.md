# CAE仿真三阶段

**what**：
前处理   ───────►   求解   ───────►   后处理
"造模型"              "算力、跑仿真"   "可视化、看结果"

## 前处理

**what**：
建立模型

**how**：流程
1. **几何**：几何导入 → 几何清理与简化
2. **网格**：表面网格划分（2D） → 体网格划分（3D） → 网格质量检查
3. **工况**：赋材料与定义连接 → 施加边界条件与载荷

### 表面网格和体网格

**what**：表面网格
（Surface Grid / vtkPolyData）
是 2D 网格，通常由三角形、四边形组成，定义了体网格的外部边界范围和形状

**what**：体网格
（Volume Grid / vtkUnstructuredGrid） 
是 3D 网格，通常由四面体、六面体组成，将 表面网格 所定义的实体空间内部划分成无数个小单元，让求解器能对每个单元进行数值计算

### node、elements、component

**what**：node
点，用于构成 elements

**what**：elements
单元，由 node 组成。对应 表面网格 或 体网格

**what**：component
容器。归类存放同类型 node、elements

## 求解器

**what**：
数值计算

**how**：流程
读模型 → 建矩阵 → 加载荷边界 → 解方程 → 得到求解结果

**what**：荷边界
载荷与边界的合称

**what**：载荷
模型受到的外力

**what**：边界
对模型位移的限制

## 后处理

**what**：
可视化

**how**：流程
导入求解结果与网格 → 选参数可视化 → 生成图表或动画 → 分析验证

> 网格来自 CAE前处理阶段，包括表面网格、体网格
> 求解结果来自 CAE求解阶段

> **选参数可视化**：
> 就是从求解结果里挑选要分析的物理量，比如应力、位移、温度，然后选可视化方式，如：
> - 变形图（看结构怎么变形）
> - 应力云图（看哪里最危险）
> - 动画（看过程如何演变）
> 让这些数据直观显示在模型上


### 应力云图

**what**：
用不同颜色映射模型上**各单元**的**应力值**，展示结构应力分布、大小的可视化结果

>**应力值**：单位面积上的内力大小，反映材料承受载荷时的受力程度

>**内力**：物体受外力作用时，内部产生的抵抗变形的相互作用力
