# static

**what**：
指定修饰对象的 存储期、作用域、链接性，修饰：
  1. [[static#static局部变量|static局部变量]]
  2. [[static#static全局变量|static全局变量]]
  3. [[类#static成员变量|static成员变量]]
  4. [[static#static函数|static函数]]
  5. [[成员函数#static成员函数|static成员函数]]

## static局部变量

**what**：
1. 静态存储期，默认初始化为 0

## static全局变量

**what**：
1. 静态存储期，默认初始化为 0
2. 内部链接性

## static函数

**what**：
1. 内部链接性
