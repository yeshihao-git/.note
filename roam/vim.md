# vim
## 基础使用

普通模式 -> 插入模式
```bash
o      # 在下⽅插⼊⼀⾏
O      # 在上⽅插⼊⼀⾏
a      # 在当前光标后插⼊
i      # 在当前光标前插⼊
A      # NOTE 在⾏尾插⼊
I      # NOTE 在⾏⾸插⼊
C      # NOTE 删除到⾏尾并进入插入模式
```

光标移动
```bash
hjkl # 上下左右
gg   # 移动到第⼀⾏
G    # 移动到最后⼀⾏/指定⾏
w    # 移动到下⼀个单词的开头
e    # 移动到下⼀个单词的结尾
b    # 移动到上⼀个单词的开头
0    # 移动到⾏⾸
^    # 移动到第⼀个⾮空字符
$    # 移动到⾏尾
%    # NOTE 移动到匹配的括号处
gd   # 移动到变量定义处
```

复制/粘贴/删除
```bash
y  # 复制
yy # 复制一行
p  # 粘贴
x  # 删除字符
dd # 删除整⾏
D  # 删除到⾏尾
```

搜索/替换
```bash
/                         # 搜索下⼀个
?                         # 搜索上⼀个
​# / *                     # NOTE 快速搜索当前单词
r                         # 替换单个字符
R                         # 持续替换字符
~                         # NOTE 修改⼤⼩写
:[range]s/from/to/[flags] # 将 range 范围内的 from 替换为 to
```

撤销/重做
```bash
u   # 撤销
C-r # NOTE 重做
```

宏
```bash
# ⽣成从 1 到 1000 的序列
q[abcd] # NOTE 录制宏到寄存器a/b/c/d
q       # NOTE 停止录制
@[abcd] # NOTE 回放宏a/b/c/d
:reg [abcd] # 查看寄存器a/b/c/d中存储的宏操作
```

窗口管理
```bash
:sp        # 横向分屏
:vsp       # 纵向分屏
C-w [hjkl] # 切换窗口
C-w [HJKL] # 改变窗口方向（H：左、J：下、K：上、L：右）
C-w q      # 关闭窗口
C-w w      # 循环切换窗口
```

标记
```bash
m  # ⽣成标签
`  # 跳转
```

可视模式
```bash
v   # 进入可视模式
V   # 选中一行进入可视模式
C-v # 选中块进入可视模式
```

操作系统
```bash
:e .       # 打开当前目录
:cd <目录> # 进入目录
```

退出
```bash
:wq # 保存并退出
:q! # 强制退出
```

## leetcode vim模式复制文本到系统剪切板

1. 鼠标选中
2. Ctrl + Insert

## 多光标批量处理文本

1. `Ctrl + v`进入可视化模式
2. `j`/`k`选中要处理的多行
3. `Shift + i`进入插入模式
4. 输入内容
5. `Esc`后应用内容
