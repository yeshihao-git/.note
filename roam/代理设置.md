# 代理设置
## linux 终端走代理

```bash
export http_proxy="http://127.0.0.1:7890"
export https_proxy="http://127.0.0.1:7890"
curl -I google.com # 测试代理是否成功
```

## git 走代理

[[git#git|git]] 不会自动继承系统环境变量，需要单独设置 git 代理
```bash
git config --global http.proxy "http://127.0.0.1:7890"   # 设置http代理
git config --global https.proxy "http://127.0.0.1:7890"  # 设置https代理
git config --global --list                               # 检查是否设置成功
```

## 同一局域网内连另一台机器的代理

1. 机器A 和 机器B 在 同一局域网
2. 在 A 的 clash 中 打开局域网连接
3. B 以 linux 终端走代理 的方式连接 A
	- `IP`：A 的 IP
	- `PORT`：A 中的 clash 的端口
