# 关键字

**what**：
C++中保留的标识符，有特殊含义，不能被用作标识符

## 说明符

**what**：
关键字的子集，用于说明变量/函数特性。分为：
1. 存储类说明符
2. 函数说明符
3. 访问说明符
4. 其他说明符

**what**：存储类说明符
说明 对象的 存储期、链接性

| 说明符            | 含义                   | 注释               |
| -------------- | -------------------- | ---------------- |
| [[extern]]     | 静态存储期，外部链接性          |                  |
| [[static]]     | 静态存储期，内部链接性          |                  |
| thread_local   | 变量存储期和线程绑定           |                  |
| mutable        | 即使是 const，也允许修改      |                  |
| [[类型推导\|auto]] | 自动存储期                | C++11 后不属于存储类说明符 |
| register       | 自动存储期 + 提示编译器将其放入寄存器 | C++17 后不属于存储类说明符 |

**what**：函数说明符 
说明 函数特性

| 说明符                         | 含义           |
| --------------------------- | ------------ |
| [[inline]]                  | 内联           |
| virtual                     | 虚函数、纯虚函数、虚基类 |
| [[构造函数#explicit\|explicit]] | 禁止隐式转换       |

**what**：访问说明符
说明 类成员访问权限

|说明符|含义|
|---|---|
|public|公有|
|private|私有|
|protected|保护|

### 类型限定符

**what**：
说明符的子集，用于限制 访问/修改 规则
1. 🔥 是类型的一部分 （eg：int 和 const int 是不同类型）

| 说明符      | 含义         |
| -------- | ---------- |
| const    | 常量，值不可修改   |
| volatile | 易变，防止编译器优化 |
| restrict | C 语言的类型限定符 |
