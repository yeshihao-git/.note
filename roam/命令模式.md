# 命令模式

**what**：
一种行为设计模式，将一次操作请求封装为独立命令对象，隔离调用方与接收方（执行方），对外提供统一执行接口（`execute`）
1. 可扩展实现 ==撤销、队列、日志== 等功能

![[Pasted image 20260617200411.png]]

**how**：
1. 抽象命令 派生 多个具体命令
2. 命令调用者 依赖 抽象命令，提供命令执行接口（`execute`）
3. 具体命令 依赖 命令接收者

```cpp
#include <string>
#include <iostream>
#include <memory>
#include <vector>

// 命令接收者
class Receiver
{
public:
    void use(std::string params)
    {
        std::cout << "receiver is using " << params << std::endl;
    }
};

// 命令
class Command
{
public:
    virtual ~Command() = default;
    virtual void execute() = 0;
};

// 复杂命令
class ConcreteCommand1 : public Command
{
public:
    ConcreteCommand1(std::unique_ptr<Receiver> receiver, std::string params) : m_receiver(std::move(receiver)), m_params(params) {}
    void execute() override
    {
        m_receiver->use(m_params);
    }

private:
    std::unique_ptr<Receiver> m_receiver;
    std::string m_params;
};

// 简易命令
class ConcreteCommand2 : public Command
{
public:
    void execute() override
    {
        std::cout << "ConcreteCommand2" << std::endl;
    }
};

// 命令调用者（发送者）
class Invoker
{
public:
    void setCommand(std::unique_ptr<Command> command)
    {
        m_command = std::move(command);
    }

    void executeCommand()
    {
        m_command->execute();
    }

private:
    std::unique_ptr<Command> m_command;
};

// 客户端代码
void clientCode()
{
    std::unique_ptr<Invoker> invoker = std::make_unique<Invoker>();
    std::unique_ptr<Receiver> receiver = std::make_unique<Receiver>();
    invoker->setCommand(std::move(std::make_unique<ConcreteCommand1>(std::move(receiver), "something")));
    invoker->executeCommand();

    invoker->setCommand(std::move(std::make_unique<ConcreteCommand2>()));
    invoker->executeCommand();
}

int main()
{
    clientCode();
}
```
