# 咖啡冲煮指南
## 咖啡萃取

**what**：萃取
咖啡中的风味物质转移到水中的过程。这些风味物质的溶解顺序：
1. 水果酸：酸味、花香、果香
2. 美拉德产物和焦糖：甜感、坚果、谷物等风味
3. 干馏物质：苦味、烟熏、炭味

通过控制这三种风味物质的溶解占比，得到一杯偏酸/甜/苦的咖啡

### 水果酸

**what**：水果酸
咖啡生豆中的有机酸，常见种类如柠檬酸（对应柑橘类水果）、苹果酸（对应苹果等），可引发不同水果风味联想，提供花香、果香

影响咖啡酸度的关键因素：
1. 产地：高海拔地区的咖啡豆能积累更多有机酸
2. 处理法：水洗处理让酸感更纯净清晰，日晒处理带来更复杂的水果酸香，蜜处理介于两者之间
3. 烘焙度：有机酸易受热挥发，浅烘咖啡豆保留的酸更多，深烘则酸感减弱

选酸感明显咖啡：优先选择高海拔产地、浅度烘焙、水洗处理的咖啡豆

### 美拉德产物和焦糖

**what**：美拉德产物和焦糖
美拉德反应产物，是咖啡豆中的氨基酸和还原糖，在 120℃-170℃烘焙时生成的，提供甜感、坚果、麦芽、谷物等风味（美拉德反应先发生）；焦糖是焦糖化的产物，是咖啡豆中的蔗糖、葡萄糖等糖类，在 170℃以上时分解生成的，提供更浓郁的焦糖甜感与轻微焦味（焦糖化反应后发生）。构成咖啡萃取中段的核心甜感。

### 干馏物质

**what**：干馏物质
在咖啡烘焙的最后阶段，由热量分解咖啡豆的木制纤维而产生，提供苦味、烟熏、炭味

干馏物质分子大，需要更长时间或更高温才能溶解，萃取时间过长/萃取温度过高容易导致干馏物质大量溶解，造成咖啡苦涩

通过 研磨度、水温 控制萃取时间，减少干馏物质溶解

## 咖啡味道与口感

咖啡的味道由 萃取率 和 养豆时间 决定，口感由 浓度 决定

### 萃取率

**what**：萃取率
咖啡粉中有多少风味物质溶解到水中，决定了咖啡的味道（酸甜苦）
咖啡的味道（酸甜苦）由 萃取率 决定，萃取太快，咖啡偏酸，萃取过慢，咖啡偏苦涩。理想的萃取率为 18%~22%，低于18%时萃取不足，咖啡偏酸，高于22%时萃取过度，咖啡偏苦涩

通过 粉水比、研磨度、水质、水温 这四个萃取变量可以控制 萃取率 ，达到酸苦平衡的效果
1. 粉水比：理想 1:15~1:17
2. 研磨度：颗粒粗细
3. 水质：过滤氯，含有适量矿物质（镁、钙）
4. 水温：理想 90.5°C~96°C

### 养豆时间

**what**：养豆 
咖啡豆排二氧化碳的过程

直接冲煮未养豆的咖啡，二氧化碳过多，会阻碍水和咖啡粉的接触，导致萃取不均衡，产生苦涩味
```养豆时间公式
25度 = 7 x （1 + 0.5 x N） = 7天

N = 0
以25度为基准，每降低5度，N + 1
养豆天数 = 7 x (1 + 0.5 x N)

N = 0
以25度为基准，每升高5度，N + 1
养豆天数 = 7 / (1 + 0.5 x N)
```

### 浓度

**what**：浓度
咖啡液体中有多少可溶性固体，决定了咖啡的口感（寡淡/醇厚）   
咖啡的口感（寡淡/醇厚）由 浓度 决定。理想的浓度为 1.15%~1.35%，低于1.15%时偏寡淡，高于1.35%时偏粘腻

通过 粉水比 控制 浓度 

## 咖啡豆
### 品种

**what**：阿拉比卡
精品咖啡的主流，生长在高海拔，风味复杂、酸质明亮，咖啡因含量低，产量低，价格高。细分品种：
1. 铁皮卡
2. 波旁
3. 瑰夏

**what**：罗布斯塔
生长在低海拔，有泥土、轮胎味，咖啡因含量高（阿拉比卡的2-3倍），产量高，价格低

### 产地

产地海拔越高，咖啡豆酸质越明亮，风味越复杂，三大产地及所产咖啡豆特点如下：
- 非洲（埃塞俄比亚、肯尼亚）：偏酸，水果花香
- 美洲（巴西、哥伦比亚、巴拿马）：酸苦平衡
- 亚洲（苏门答腊、也门）：醇厚（酸低，口感厚重）

### 处理法

- 水洗：先去皮，泡水里发酵后洗净晒干。酸质明亮、口感干净
- 日晒：带皮直接晒。甜感浓，有莓果、酒香
- 蜜处理：去皮但留层果胶再晒。甜感介于两者之间，有热带水果味。分为：
	1. 黑蜜：保留果胶最多，遮光布盖住晒。甜感最浓
	2. 红蜜：保留果胶中等，正常晒。甜感适中
	3. 黄蜜：保留果胶最少，晒前冲掉一点果胶。甜感最低

### 烘焙

**what**：烘焙 
将生豆变为有风味的熟豆的过程，核心是通过控制 “温度与时间” 的烘焙曲线，决定咖啡豆偏向 “豆子原产地风味” 还是 “烘焙工艺风味”。烘焙工艺分为：
1. 浅烘：豆子原产地风味，酸质明显，花香果香
2. 中烘：甜感突出
3. 深烘：烘焙工艺风味，偏苦，烟熏，醇厚度高
