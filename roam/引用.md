# 引用

**what**：
给一个对象起 别名 ，必须初始化，后续不能改变引用的对象。分为：
  1. 左值引用
  2. 右值引用

## 左值

**what**：
有名字、有地址、持久值

### 左值引用

**what**：
`type&` ，给左值起 别名 ，必须初始化，后续不能改变引用的对象。分为：
  1. const左值引用
  2. 非const左值引用
  3. 悬空引用

#### const左值引用

const左值引用能绑定
1. 左值
2. const左值
3. 右值（ 为什么可以绑定右值？创建临时对象，用右值初始化它，引用绑定该临时对象，临时对象的生命周期 延长到和引用一致）
```cpp
/* 常量 */
const int a = 10;
const int& ra = a;           // 合法：绑定 const左值（常量）

/* 非常量 */
int b = 20;
const int& rb = b;           // 合法：绑定 普通左值（非常量）
const int c = 30;
// int& rc = c;              // FIXME：非常量引用不能绑定到常量

/* 右值 */
const int& ref1 = 100;       // 合法：绑定 字面量100
const int& ref2 = 20 + 30;   // 合法：绑定 表达式生成的临时对象
// int& ref3 = 100;          // FIXME：普通引用不能绑定临时对象
```

#### 非const左值引用

非const左值引用能绑定左值，不能绑定 右值 、 const左值

> 为什么不能绑定？
> 核心在于 非const左值引用要求被引用对象可修改，而 右值、const左值 都不可修改

#### 悬空引用

**what**：
被引用对象销毁，则引用变为悬空引用 

## 右值

**what**：
无名字、无地址、临时值（eg：字面量、函数返回临时值、表达式生成的临时对象）
  1. 可以激活 移动语义
  2. 通过 右值引用 绑定，并修改值
  3. 当左值出现在右边时，通过 平凡转换 左值隐式转换为右值

### 右值引用

**what**：
（cpp11） `type&&` ，对右值的引用，只能绑定右值， 延长右值生命周期 与右值引用一致，用于 移动语义 和 完美转发
  1. 本质 是 左值
  2. 函参中的 右值引用 可以参与 函数重载
  3. 可以通过 非const右值引用 修改右值
```cpp
void fun(const int& lref) { // 左值引用版本
  std::cout << "l-value reference to const: " << lref << '\n';
}

void fun(int&& rref) {      // NOTE 右值引用版本，参与函数重载
  std::cout << "r-value reference: " << rref << '\n';
}

int main() {
  int x{ 5 };
  int& ref_l {x}; // 左值引用 绑定 左值x
  int&& ref {5};  // 右值引用 绑定 右值5
  ref = 6;        // NOTE 通过 非const右值引用 修改右值

  fun(ref);       // NOTE 右值引用 是 左值，调用 左值引用版本
  fun(x);         // 调用 左值引用版本
  fun(ref_l);     // 调用 左值引用版本
  fun(5);         // 调用 右值引用版本
}

// result：
// l-value reference to const: 6
// l-value reference to const: 5
// l-value reference to const: 5
// r-value reference: 5
```

#### 万能引用

**what**：
`T&&` ，使 函数模板可以接收左/右值 的引用，并根据 引用折叠 确定最终类型
  1. 等于 右值引用 + 类型模板参数

##### 引用折叠

**what**：
多个引用组合后简化为 右值引用

规则 ：右值引用的右值引用 折叠为 右值引用，其他情况为 左值引用

|组合方式|折叠结果|
|---|---|
|T& &|T&|
|T& &&|T&|
|T&& &|T&|
|T&& &&|T&&|

##### 完美转发

**what**：
（cpp11） `std::forward` 用于保持 函参的左右值属性

**why**：
以万能引用为形参的函数模板A 调用 函数B，万能引用捕获的参数传入B，B只会调用左值版本，原因在于 右值引用是左值 ，解决方式：完美转发

## 引用限定符

**what**：
使用 `&` 、 `&&` 分别限定函数只能用于左值或右值，用于 阻止旧标准可以对右值赋值
  1. 可参与 函数重载
