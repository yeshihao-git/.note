# 流程控制

**what**：
编程语言 通过 特定语句 控制程序执行顺序 ，分为：
  1. 选择结构
  2. 循环结构
  3. 跳转结构

## 选择结构

**what**：
根据条件判断，执行不同逻辑
1. if-else
2. switch-case

### switch-case

**what**：
switch 基于 整型/枚举 匹配 case块，实现多分支选择执行
1. 使用 break 或 return 避免分支 穿透执行（case块 缺少 break 或 return，继续执行后面的 case块）
```cpp
int main() {
    switch (2) {
    case 1: // Does not match
        std::cout << 1 << '\n'; // Skipped
    case 2: // Match!
        std::cout << 2 << '\n'; // Execution begins here
    case 3:
        std::cout << 3 << '\n'; // This is also executed
    case 4:
        std::cout << 4 << '\n'; // This is also executed
    default:
        std::cout << 5 << '\n'; // This is also executed
    }
}
```
2. default 处理未匹配情况
3. 所有case块 共享 switch作用域
```cpp
switch(整型/枚举) {
    case 值1: // 类似 if(整型/枚举 == 值1)
      语句;
      break;
    case 值2: // 所有case块共享 switch作用域，除非 case块 使用 {}
      语句;
      break;
    default:  // 处理未匹配情况
      语句;
      break;
}
```

## 循环结构

**what**：
条件满足时，重复执行某段逻辑
1. for
2. while
3. do-while

## 跳转结构

**what**：
主动改变程序执行顺序
1. break
2. continue
3. return
4. goto
