# 策略模式

**what**：
一种行为设计模式， 它能让你定义一系列算法， 并将每种算法分别放入独立的类中， 以使算法的对象能够相互替换

**how**：
![[Pasted image 20260618114102.png]]

```cpp
#include <string>
#include <iostream>
#include <memory>

class Strategy
{
public:
    virtual ~Strategy() = default;
    virtual void execute() = 0;
};

class ConcreteStrategyA : public Strategy
{
public:
    void execute() override
    {
        std::cout << "ConcreteStrategyA" << std::endl;
    }
};

class ConcreteStrategyB : public Strategy
{
public:
    void execute() override
    {
        std::cout << "ConcreteStrategyB" << std::endl;
    }
};

class ConcreteStrategyC : public Strategy
{
public:
    void execute() override
    {
        std::cout << "ConcreteStrategyC" << std::endl;
    }
};

class Context
{
public:
    void setStrategy(std::unique_ptr<Strategy> strategy)
    {
        m_strategy = std::move(strategy);
    }

    void doSomething()
    {
        if (m_strategy)
        {
            m_strategy->execute();
        }
    }

private:
    std::unique_ptr<Strategy> m_strategy;
};

void clientCode()
{
    auto context = std::make_unique<Context>();
    auto strategyA = std::make_unique<ConcreteStrategyA>();
    auto strategyB = std::make_unique<ConcreteStrategyB>();
    auto strategyC = std::make_unique<ConcreteStrategyC>();

    context->setStrategy(std::move(strategyA));
    context->doSomething();

    context->setStrategy(std::move(strategyB));
    context->doSomething();

    context->setStrategy(std::move(strategyC));
    context->doSomething();
}

int main()
{
    clientCode();
}
```
