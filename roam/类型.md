# 类型

**what**：
告诉 编译器 如何 解释内存中的数据
+ 位：内存组成的单位 ，每个位存 0或1
+ 字节：内存访问的单位 ，每个字节有唯一地址， 1字节 = 8位

数据类型总结如下

| 类型     | 含义                        | 示例                                |
| ------ | ------------------------- | --------------------------------- |
| 基本数据类型 | cpp 内置的类型                 | int                               |
| 复合数据类型 | 基于现有类型定义的类型               | int&、double*、std::string、Fraction |
| 用户定义类型 | 类类型 或 枚举（包含 标准库、编译器扩展实现）  | std::string、Fraction              |
| 程序定义类型 | 类类型 或 枚举（不包含 标准库、编译器扩展实现） | Fraction                          |

## 基本数据类型

**what**：
内置的类型（eg：int、double、bool等）

### 基本数据类型大小（32位系统、64位系统）

|数据类型|32 位系统（字节）|64 位系统（字节）|说明|
|---|---|---|---|
|char|1|1|1 字节|
|short|2|2|2 字节|
|int|4|4|32 位和 64 位系统中多数为 4 字节|
|long|4|8|32 位系统中 4 字节，64 位系统中 8 字节|
|long long|8|8|至少 8 字节，32 位和 64 位系统中均为 8 字节|
|float|4|4|单精度浮点数，固定 4 字节|
|double|8|8|双精度浮点数，固定 8 字节|

> long的大小是核心差异

> 32位和64位系统 :: 指 CPU 一次处理能 32位（4字节）/64位（8字节） 的数据

### char

**what**：
一种类型，表示字符，大小为 1字节
1. 底层：编译器将 1字节 数据解释为 char 类型，具体来说：就是将这1个字节内存储的数字 通过 ASCII 对应到 字符
2. char 和 int 之间很方便互相转换

## 复合数据类型

**what**：
基于现有类型定义的类型

复合数据类型：
1. 函数[^1]
2. 数组
3. 指针
4. 引用
5. 枚举
6. 类（类、结构体、联合体）

## 用户定义类型

**what**：
非cpp定义的 类类型 或 枚举（包含 标准库、编译器扩展实现），使用时，编译器需看到 完整定义（不能只是前向声明），为 该类型对象分配内存

## 程序定义类型

**what**：
非cpp定义的 类类型 或 枚举（不包含 标准库、编译器扩展实现）






[^1]: void func(int x, double y) 的类型是 void(int, double)
