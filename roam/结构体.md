# 结构体

**what**：
用户自定义的 复合数据类型，由 成员变量 和 成员函数 封装而成
1. 访问权限 、 继承权限 默认为 public
2. 最核心的使用场景：作为 聚合体 使用

## 聚合体

**what**：
将多种类型的成员变量封装在一起（专注于 数据聚合 而非封装）
```cpp
struct WindowConfig { // 常见使用场景：数据存储 -> 配置
    int width;
    int height;
    bool fullscreen;
    const char* title;
};
WindowConfig config = {800, 600, false, "My App"};
CreateWindow(config);
```
1. 支持 {} 聚合初始化 （非聚合体 使用 构造函数），常用载体是 struct
2. 与 C语言兼容的 数据结构
3. 用于 数据存储
4. constexpr 聚合体，提高运行性能
```cpp
struct RGB {
    uint8_t r, g, b;
};

// 必须初始化
constexpr RGB colors[] = {
    {255, 0, 0},   // 红
    {0, 255, 0},   // 绿
    {0, 0, 255}    // 蓝
};
```

有哪些？
1. C风格数组
2. 类类型（结构体、类、联合体）
   需满足以下条件：
	1. 无 private/public的 非static成员变量
	2. 无 引用类型的 非static成员变量
	3. 无 用户声明的构造函数
	4. 无 用户声明的拷贝/移动赋值运算符
	5. 无 虚函数、虚基类
```cpp
class Date {
public:                // private/public 不能使用聚合初始化
    int m_year {};
    int m_month {};
    const int m_day {};
    static int s_day;
    // int& m_hour;    FIXME 引用类型 不能使用聚合初始化

    void print() const {
        cout << m_year << " " << m_month << " " << m_day << " " << s_day << endl;
    }
};

int Date::s_day {1};

int main() {
    Date today { 1997, 10, 22};
    today.print();
}
```
