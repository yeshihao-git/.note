# 运算符

**what**：
具有特定语法格式、语义的符号

## 下标运算符

**what**：
`operator[]` ，
1. 重载后使类对象能像数组一样通过 `[]` 语法 访问内部元素
2. 下标运算符返回的是 引用
```cpp
int arr[10] = {1,2,3,4,5,6,7,8,9,0};
int &p = arr[3];
```

## 逗号运算符

**what**：
从左到右计算，返回最右边的计算结果
```cpp
int a = 1, b = 2;
int z;
z = (a, b);       // 计算 a，再计算 b，将 b 的值赋给 z
z = a, b;         // 计算 a，将 a 值赋给 z ，再计算 b
for (int c = 3, d = 4; ...; ...) {...}
```

## 赋值运算符

**what**：
`operator=`，将一个对象的内容赋值给另一个对象，分为：拷贝赋值运算符、移动赋值运算符
1. 赋值表达式的返回 引用，提高效率
2. 支持 链式表达式
```cpp
int a = 3, b = 4;
decltype(a = b) d = a; // NOTE 返回引用：d的类型为int &，值为3
a = b = 1;             // NOTE 支持链式表达式：等价于 a = (b = 1)，将 b=1 的返回值赋给 a

// NOTE 返回引用，提高效率
Obj a, b;
a = b = c; // 若非引用，则b=c的结果是拷贝复制到返回值，再赋给a
           // 若为引用，则返回值直接引用b=c的结果
```

### 拷贝赋值运算符

**what**：
`operator=` ，更新一个已存在的对象（对象已创建，用另一个对象来赋值）
```cpp
// 拷贝赋值
STU stu1;        // 默认构造
STU stu2;
stu2 = stu1;     // NOTE 调用 拷贝赋值运算符（stu2已存在）
```

```cpp
// 拷贝构造
STU stu1;        // 默认构造
STU stu2 = stu1; // 调用 拷贝构造（stu2不存在）
```

1. 检查并处理自赋值 ：赋值需要删除原对象，再将另一个对象赋值过来，若是自赋值，会删除自己
```cpp
MyString& MyString::operator= (const MyString& str) {
	// NOTE 1. 自赋值处理：直接返回
  if (this == &str)
    return *this;

  // 2. 删除自己的值
  if (m_data) delete[] m_data;

  m_length = str.m_length;
  m_data = nullptr;

  // 3. 复制他人的值
  if (m_length)
    m_data = new char[static_cast<std::size_t>(str.m_length)];

  std::copy_n(str.m_data, m_length, m_data);

  // 4. return the existing object so we can chain this operator
  return *this;
}
```

## 运算符重载

**what**：
`operator<运算符>()` ，用于为类重新定义运算符（eg：`+`、`-`、`==`、`<<`），使类能像 基本数据类型 一样使用运算符进行操作
1. 本质 ：函数重载（根据 操作数类型）
2. 🔥 运算符重载 需要至少一个操作数是 用户定义类型 
```cpp
operator+(int, Mystring) // ok
operator+(int, double)   // FIXME
```
3. 操作数顺序不同 或 操作符之间有关联，可以使用 已实现的运算符来实现
```cpp
// NOTE 参数顺序不同
Cents operator+(const Cents& c1, int value);
Cents operator+(int value, const Cents& c1) { return c1 + value; } // 调用 operator+(const Cents& c1, int value);

// NOTE 操作符之间有关联
bool operator== (const Cents& c1, const Cents& c2) { return c1.m_cents == c2.m_cents; }
bool operator!= (const Cents& c1, const Cents& c2) { return !(operator==(c1, c2)); }
bool operator< (const Cents& c1, const Cents& c2) { return c1.m_cents < c2.m_cents; }
bool operator> (const Cents& c1, const Cents& c2) { return operator<(c2, c1); }
bool operator<= (const Cents& c1, const Cents& c2) { return !(operator>(c1, c2)); }
bool operator>= (const Cents& c1, const Cents& c2) { return !(operator<(c1, c2)); }
```
4. 运算符重载 有实现方式：
	1) 运算符重载实现：成员函数
    2) 运算符重载实现：非成员函数

### 运算符重载实现：成员函数

**what**：
1. 适合 左操作数 是 类对象 的情况（左操作数 是 隐式的 `*this` 对象） 
```cpp
class Cents {
private:
    int m_cents {};
public:
    Cents(int cents) : m_cents { cents } { }
    Cents operator+(int value) const;        // NOTE 左操作数 是 隐式的*this，成员函数重载：Cents + int
    int getCents() const { return m_cents; }
};

Cents Cents::operator+ (int value) const {
    return Cents { m_cents + value };
}

int main() {
  const Cents cents1 { 6 };
  const Cents cents2 { cents1 + 2 };
  std::cout << "I have " << cents2.getCents() << " cents.\n";
}
```

哪些运算符适合？
1. （ 必须 ）赋值 `=` 、下标 `[]` 、函数调用 `()` 、成员选择 `->` 、类型转换运算符
2. 改变对象状态的： `+=` ， `-=` ， `++` ， `--`

#### 类型转换运算符

 **what**：
`operator<type> 类` -> 基本类型 （隐式转换）

最佳实践
优先使用 转换构造函数

`operator<type>()` 书写方式：
1. 必须 非static，应该 const（以便用于 常量对象）
2. 无 函参
3. 无 返回
4. 一般设置为 `explicit` 防止隐式转换
```cpp
class Cents {
private:
    int m_cents{};
public:
    Cents(int cents=0) : m_cents{ cents } { }

    // NOTE int类型转换运算符重载：非static，为const，无函参，无返回
    operator int() const { return m_cents; }

    int getCents() const { return m_cents; }
    void setCents(int cents) { m_cents = cents; }
};

// 输入 Cents类对象 -> 重载int类型转换
void printInt(int value) { std::cout << value; }

int main() {
    Cents cents{ 7 };
    printInt(cents);
    std::cout << '\n';
}
```

#### 自增运算符

**what**：
`operator++`
1. 前置++： `operator++()` （`operator--`同理）
2. 后置++： `operator++(int)` 
```cpp
class Digit {
private:
    int m_digit{};
public:
    Digit(int digit=0) : m_digit{digit} { }
    // NOTE 前置++/--
    Digit& operator++();
    Digit& operator--();
    // NOTE 后置++/--
    Digit operator++(int);
    Digit operator--(int);

    friend std::ostream& operator<< (std::ostream& out, const Digit& d);
};

Digit& Digit::operator++() {
    if (m_digit == 9)
        m_digit = 0;
    else
        ++m_digit;

    return *this;
}

Digit& Digit::operator--() {
    if (m_digit == 0)
        m_digit = 9;
    else
        --m_digit;

    return *this;
}

// NOTE 后置实现方式
Digit Digit::operator++(int) {
    Digit temp{*this};

    ++(*this); // 调用 前置++

    // 返回临时对象
    return temp;
}

Digit Digit::operator--(int) {
    Digit temp{*this};

    --(*this);

    return temp;
}

std::ostream& operator<< (std::ostream& out, const Digit& d) {
	out << d.m_digit;
	return out;
}

int main() {
    Digit digit { 5 };

    std::cout << digit;
    std::cout << ++digit; // calls Digit::operator++();
    std::cout << digit++; // calls Digit::operator++(int);
    std::cout << digit;
    std::cout << --digit; // calls Digit::operator--();
    std::cout << digit--; // calls Digit::operator--(int);
    std::cout << digit;
}
```

### 运算符重载实现：非成员函数

**what**：
非成员函数（全局函数、友元函数）：
1. 适合 左操作数 不是类对象 的情况
2. 访问类内 private/protected 成员 使用 类公共接口 或 友元函数
```cpp
// 使用类提供的公共接口访问私有成员
class Cents {
private:
  int m_cents{};
public:
  Cents(int cents) : m_cents{ cents } { }
  int getCents() const { return m_cents; }
};

// NOTE 普通函数：使用类提供的公共接口访问私有成员
Cents operator+(const Cents& c1, const Cents& c2) {
  return Cents{ c1.getCents() + c2.getCents() };
}

int main() {
  Cents cents1{ 6 };
  Cents cents2{ 8 };
  Cents centsSum{ cents1 + cents2 };
  std::cout << "I have " << centsSum.getCents() << " cents.\n";
}
```

```cpp
// 使用友元函数访问私有成员
class Cents {
private:
	int m_cents {};
public:
  Cents(int cents) : m_cents{ cents } { }
  friend Cents operator+(const Cents& c1, const Cents& c2); // NOTE 友元函数：类外实现 Cents + Cents
  int getCents() const { return m_cents; }

  // NOTE 友元函数：类内实现 -> 类内实现也属于 类的成员
  // friend Cents operator+(const Cents& c1, const Cents& c2)
	// {
	//    return Cents { c1.m_cents + c2.m_cents };
	// }
};

// 友元函数：类外实现
Cents operator+(const Cents& c1, const Cents& c2) {
	return c1.m_cents + c2.m_cents;
}

int main() {
	Cents cents1{ 6 };
	Cents cents2{ 8 };
	Cents centsSum{ cents1 + cents2 };
	std::cout << "I have " << centsSum.getCents() << " cents.\n";
}
```

哪些运算符适合？
1. （ 必须 ）左操作数是流对象： `<<` ， `>>` 
```cpp
class Point
{
private:
    double m_x{};
    double m_y{};
    double m_z{};

public:
    Point(double x=0.0, double y=0.0, double z=0.0)
      : m_x{x}, m_y{y}, m_z{z}
    {
    }

    friend std::ostream& operator<< (std::ostream& out, const Point& point);
};

std::ostream& operator<< (std::ostream& out, const Point& point)
{
    out << "Point(" << point.m_x << ", " << point.m_y << ", " << point.m_z << ')';
    return out; // 返回 std::ostream 所以我们可以链式调用
}

int main()
{
    const Point point1 { 2.0, 3.0, 4.0 };

    std::cout << point1 << '\n';

    return 0;
}
```
2. 具有对称性，支持隐式类型装换： `+` ， `-` ， `==` ， `!=`
