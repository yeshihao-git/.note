# 适配器模式

**what**：
一种结构型设计模式， 它能使接口不兼容的对象能够相互合作。两种实现方式：
1. **对象适配器**：适配器实现了其中一个对象的接口， 并对另一个对象进行封装（所有流行的编程语言可用）
2. **类适配器**：适配器同时继承两个对象的接口（仅支持多重继承的编程语言中实现， 例如 C++）

## 对象适配器

![[Pasted image 20260617103916.png|557]]
**how**：
适配器 依赖 服务端，并继承 客户端接口类 重写接口方法，接口方法中存在对服务端方法的调用

```cpp
#include <string>
#include <algorithm>
#include <iostream>
#include <memory>

// Client Interface
class Target 
{
public:
    virtual ~Target() = default;
    virtual std::string request() const
    {
        return "Target: default behavior";
    }
};

// Service
class Adaptee
{
public:
    std::string specificRequest() const
    {
        return ".eetpadA eht fo roivaheb laicepS";
    }
};

// Adapter
class Adapter : public Target
{
public:
    Adapter(std::unique_ptr<Adaptee> adaptee) : m_adaptee(std::move(adaptee))
    {
    }
    std::string request() const override
    {
        std::string specificString = m_adaptee->specificRequest();
        std::reverse(specificString.begin(), specificString.end());
        return "Adapter: (TRANSLATED) " + specificString;
    }

private:
    std::unique_ptr<Adaptee> m_adaptee;
};

// Client
void clientCode(std::unique_ptr<Target> target)
{
    std::cout << target->request() << std::endl;
}


int main()
{
    std::unique_ptr<Target> target = std::make_unique<Target>();
    clientCode(std::move(target));
    
    std::unique_ptr<Adaptee> adaptee = std::make_unique<Adaptee>();
    std::unique_ptr<Adapter> adapter = std::make_unique<Adapter>(std::move(adaptee));
    clientCode(std::move(adapter));
}
```
