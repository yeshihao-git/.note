# 序列化
**what**：
将内存中的 对象/数据结构 转成 可存储/传输 的 字符串/字节串。常见的有：
1. [[#JSON]]
2. XML
3. 二进制流

**why**：
内存中的对象是临时的，序列化后才能存到 文件、数据库 或 网络传输到其他设备

## JSON
**what**：
一种数据序列化的具体格式。它的表示如下：
1. 数组 用 `[ ... ]`包裹
2. 对象 用 `{ ... }`包裹
3. 项与项之间用`,`隔开
4. 对象中的键值对用`:`隔开
```json
{
    "FirstName": "John",
    "LastName": "Doe",
    "Age": 43,
    "Address": {
        "Street": "Downing Street 10",
        "City": "London",
        "Country": "Great Britain"
    },
    "Phone numbers": [
        "+44 1234567",
        "+44 2345678"
    ]
}
```
> **避坑**：一个标准的`JSON`文件里，最外层只能有一个`{}`或`[]`

**why**：
便于阅读和编写
